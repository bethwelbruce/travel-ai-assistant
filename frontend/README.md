# Travel AI Assistant

Full-stack Q&A system with DeepSeek integration

## Features
- Real-time travel advice
- Query history (local storage)
- Responsive design

## Setup

### Backend
1. Create `.env` file:
DEEPSEEK_API_KEY=your_api_key

2. Install dependencies:
```bash
pip install -r requirements.txt
Run:

bash
